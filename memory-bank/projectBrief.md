# Project Brief - RICOH360 Business Pack Onboarding

## Purpose
Structured online platform for onboarding, support, and knowledge base for RICOH360 Business Pack Series customers, prospects, and internal teams.

## Problem
Content exists but is dispersed. No centralized, shareable platform for client onboarding, post-sale support, or pilot accompaniment.

## Solution
VitePress-based bilingual (EN/FR) documentation site, deployed on GitHub Pages.

## Target audiences
1. Prospects: understanding Business Pack Series
2. Pilots/Tests: onboarding during test phases
3. Active clients: usage support, team onboarding
4. Client managers: internal team transmission

## Content source
48 markdown files in _Shared_Knowledge, of which 28 are client-facing guides already structured by user journey.

## Phases
- V1: English + French KB online, searchable, on GitHub Pages
- V2: Password-gate, role-based navigation, UX improvements
- V3: Analytics, internal section, feedback widget, content evolution

## Stack
VitePress 1.x, GitHub Pages, GitHub Actions CI/CD

## Constraints
- Bilingual FR/EN mandatory
- Brand: #3532CA blue, #2A303C black, B2B professional tone
- EMEA positioning: co-construction, not official launch
- No AI references in any visible output
