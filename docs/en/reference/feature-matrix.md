## Key Differences

- **Web App is the full management platform.** All organization, editing, AI processing, annotation, sharing, and administration features are exclusive to the Web App.
- **Mobile App is capture-focused.** It controls the camera, triggers capture, manages WiFi configuration, and provides basic gallery viewing. It does not support annotations, AI features, or administration.
- **Simple Capture is THETA X only.** The guided album-based capture workflow requires the THETA X plugin. THETA A1 does not support plugins.
- **THETA A1 excels at endurance.** It supports up to 120 minutes of video recording and IP64 protection, but lacks the touchscreen, removable battery, and plugin ecosystem of the THETA X.
- **Sharing has structural limits.** Links are per-media only (no album or project sharing), annotations are stripped from shared links, and there is no password protection or link expiration.
